# RogueKit：用于Godot引擎的模块化Roguelike开发套件

本文档是 **RogueKit** 的高级概述。**RogueKit** 是一个基于Godot 4.4.1引擎和GDScript语言的模块化、可重用Roguelike游戏库。其核心目标是抽象Roguelike游戏开发中的通用复杂性，提供一个稳固、数据驱动且易于扩展的基础框架，从而使开发者能够将精力集中在创造独特的游戏玩法、内容和体验上。
